# react-native-unified-webrtc

Unified webrtc player

## Installation

```sh
npm install react-native-unified-webrtc
```

## Usage


```js
import { UnifiedWebrtcView } from "react-native-unified-webrtc";

// ...

<UnifiedWebrtcView color="tomato" />
```
